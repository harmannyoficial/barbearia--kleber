# ✂️ Sobre o Projeto

O **BarberPro** é um sistema completo para gerenciamento de barbearias, desenvolvido com foco em:

- Agendamentos rápidos
- Controle de clientes
- Gestão financeira
- Experiência moderna e intuitiva

O projeto foi pensado para funcionar tanto no computador quanto no celular, mantendo uma interface limpa, profissional e minimalista.

# ⚙️ Funcionalidades

## 📅 Agendamentos

- Cadastro de horários
- Calendário interativo
- Confirmação rápida
- Status de pagamento
- Histórico de clientes

## 👤 Clientes

- Cadastro automático
- Histórico de serviços
- Controle de frequência
- Informações salvas no navegador

## 💰 Financeiro

- Controle de faturamento
- Valor recebido por barbeiro
- Relatórios simples
- Painel administrativo

## 👨‍💼 Funcionários

- Cadastro de barbeiros
- Controle individual
- Comissão e vendas
- Permissões administrativas

## 📱 Responsividade

- Compatível com celular
- Interface adaptável
- Navegação otimizada
